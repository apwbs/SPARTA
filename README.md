# SPARTA

**Secure Platform for Automated decision Rules via Trusted Applications**
****

SPARTA is an approach for secure, privacy-preserving, and verifiable decision support in multi-party settings. It combines Trusted Execution Environments (TEEs), InterPlanetary File System (IPFS) along with InterPlanetary Name System (IPNS), blockchains, and policy-driven fine-grained access control to execute customizable decision logic without exposing sensitive data.
